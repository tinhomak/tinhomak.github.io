1. Inline Math

You can write math inside a sentence, like the Pythagorean theorem: \(a^2 + b^2 = c^2\).

2. Display Math (Standalone)

For more complex equations, use the double-dollar sign notation:

$$x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}$$

3. Complex Calculus Example

Here is an integral for good measure:

$$\int_{a}^{b} f(x) \,dx = F(b) - F(a)$$
